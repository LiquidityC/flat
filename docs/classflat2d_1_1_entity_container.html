Detailed Description

Holds every Entity in the game. Entities registered into the EntityContainer will automatically have their update functions called and have collisionhandling performed on them if they are collidable. Registered Entity objects will be gracefully deleted when the EntityContainer is destroyed when the GameEngine exits. If you unregister an Entity you will have to dispose of it yourself in the game code.
